在海外华人 VPS 圈，DMIT 算是口碑不错的选手，价格偏中高端，但网络质量确实有保障。

## DMIT 退款政策详解

### 标准退款规则

DMIT 提供两种主要退款保障：

**1. 3 天无理由退款（全额退款）**

满足以下全部条件，可以申请全额退款：

- 订单创建时间不超过 **3 天**
- VPS 实际使用流量不超过 **30GB**
- 属于正常新开订单（非活动特价或促销限定套餐）
- 退款金额中会扣除支付手续费（如 PayPal 等平台收取的费用）

说白了：买了三天内，跑跑测试没超 30G，基本都能退。

**2. 30 天按比例退款（Pro-rata Refund）**

超过 3 天但在 30 天内，DMIT 支持按剩余服务时间比例退款。比如你买了一个月套餐，用了 10 天，那大概能退剩下 20 天的费用。

这一条对"想换套餐"的用户很有用——不用硬撑到期，可以退掉当前套餐、重新选更合适的。

---

### 哪些情况不退款？

以下情况 DMIT 通常不受理退款，要提前了解清楚：

| 情况 | 是否退款 |
|------|----------|
| 超过 30 天 | ❌ 不退 |
| 流量使用超 30GB（3 天内申请） | ❌ 不退或仅部分退 |
| 活动特价套餐（促销限定价格） | ❌ 一般不退 |
| 因违反服务条款被封号 | ❌ 不退 |
| 因用户自身操作失误（如删库、误配置） | ❌ 不退 |
| IP 被目标网站封锁 | ❌ 不属于服务故障，不退 |

特别要注意**活动特价套餐**这条——DMIT 不定期推出促销价，比节点正价便宜不少，但对应的退款政策往往更严格，有时只支持服务故障场景下的补偿，而非用户主动退款。购买前务必看清活动页面的具体说明。

### 如何申请退款？

DMIT 的退款申请流程：

1. 登录 DMIT 账户（dmit.io）
2. 进入 **Support / 客服工单** 系统
3. 新建工单，选择账单或退款相关分类
4. 说明订单号、购买日期、申请退款原因
5. 等待客服审核，一般 1-3 个工作日内处理

退款到账时间取决于支付方式：信用卡通常 5-10 个工作日，PayPal 稍快，加密货币支付的退款方式视具体情况而定。

> **小建议**：工单写清楚一点，带上订单号和使用情况，审核会快很多。含糊的申请容易被来回问，拖慢进度。

**Q：活动价格的套餐退款政策一样吗？**

不一定。DMIT 促销套餐通常会在活动页面单独注明退款条件，有时候更严格——比如不支持用户主动退款、只支持服务故障补偿。买之前务必看清楚活动说明。
